## Semantic HTML

> Use correct HTML elements for their intended purpose to improve accessibility and SEO.

```html
<!-- Good -->
<button>Submit</button>

<!-- Bad -->
<div role="button">Submit</div>
```

## ARIA Attributes

> Use ARIA attributes to enhance accessibility when semantic HTML is not sufficient.

Common ARIA attributes:

- `aria-label`: Provides an accessible name for an element.
- `aria-hidden`: Hides elements from assistive technologies.
- `aria-live`: Indicates that an element will be updated dynamically.
- `aria-expanded`: Indicates whether a collapsible element is expanded or collapsed.
- `aria-selected`: Indicates the current "selected" state of an element.
- `aria-disabled`: Indicates that an element is disabled.
- ... and so on.

## Keyboard Navigation

> Ensure that all interactive elements are accessible via keyboard.

```jsx
function Modal({ isOpen, onClose, title, children }) {
  const modalRef = useRef(null);
  const previousFocusRef = useRef(null);

  useEffect(() => {
    if (isOpen) {
      // remember what is focused before modal opened
      previousFocusRef.current = document.activeElement;
      // focus the first focusable element in the modal when it opens
      const focusable = modalRef.current.querySelectorAll('button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])');
      focusable[0]?.focus();
      return;
    }

    // restore focus to previous element when modal closes
    previousFocusRef.current?.focus();
  }, [isOpen]);

  const handleKeyDown = (e) => {
    if (e.key === "Escape") {
      onClose();
      return;
    }

    if (e.key === "Tab") {
      const focusable = modalRef.current.querySelectorAll('button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])');

      const first = focusable[0];
      if (!first) {
        e.preventDefault();
        return;
      }
      const last = focusable[focusable.length - 1];

      if (e.shiftKey) {
        // shift + tab
        if (document.activeElement === first) {
          e.preventDefault();
          last.focus();
        }
        return;
      } else if (!e.shiftKey && document.activeElement === last) {
        // tab
        e.preventDefault();
        first.focus();
      }
    }
  };

  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal" role="dialog" aria-modal="true" aria-labelledby="modal-title" ref={modalRef} tabIndex="-1" onKeyDown={handleKeyDown} onClick={(e) => e.stopPropagation()}>
        <h2 id="modal-title">{title}</h2>
        {children}
        <button onClick={onClose}>Close</button>
      </div>
    </div>
  );
}
```

## Colour and Contrast

- Minimum contrast ratios (WCAG 2.1):
  - Normal text: 4.5:1
  - Large text (18pt+ bold or 24pt+): 3:1
  - UI components and graphical objects: 3:1
- Don't convey information using colour alone
  - BAD: Red text for errors
  - GOOD: Red text + icon + descriptive text

## Images

> Use descriptive alt text for images to provide context for screen readers.

```html
<!-- Good -->
<img src="chart.jpg" alt="Revenue chart showing 20% quarterly growth in Q3" />
```
